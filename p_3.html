Energy Consumption Analysis¶

Interview project expectations could be:¶
  • complete the project analysis on a given dataset (24 to 48 hours)
  • create a PowerPoint presentation
  • present your findings to the interview committee (30 to 40 minutes)
  • answer any questions based on your analysis (15 to 30 minutes)
Data analysis skills to show in interview¶
- Understanding the requirement
- Understanding the data
- Data preprocessing
- Data exploration including visualization
- Data analysis
- Is the data sufficient?
- Significant findings 
- Actionable recommendations based on findings

- 1- What are average billing percentage of a month?

- 9- Which User ID have done wrong reading?
